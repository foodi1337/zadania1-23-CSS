Zadanie 21 – Zastosowanie identyfikatora ID w CSS

Pojęcie Znaczenie – uzupełnione przez ucznia
Co znaczy skrót ID? ID to identyfikator – unikalna nazwa elementu HTML, służąca do jego jednoznacznego oznaczenia w dokumencie.
Jaki jest cel stosowania ID? ID pozwala na indywidualne formatowanie pojedynczego elementu, nawet jeśli występuje wiele podobnych na stronie.
Jakie jest znaczenie znaku #? Znak # w CSS oznacza, że odwołujemy się do identyfikatora (ID) o konkretnej nazwie, np. #pochyly.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W HTML dodajemy do elementu atrybut id="nazwa_id".
W CSS odwołujemy się do niego przez #nazwa_id.
Przykład: <p id="pochyly">Tekst</p> a w CSS: p#pochyly { font-style: italic; }
Uwagi na temat stosowania ID (trzy konieczne wyliczenia)
  1. Każdy identyfikator ID musi być unikalny w obrębie całej strony.
  2. ID stosuje się, gdy chcemy wyróżnić pojedynczy element.
  3. Nie należy używać tego samego ID dla wielu elementów.
